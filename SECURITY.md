# Sicherheit

SafeTurn ist ein Prototyp und kein zugelassenes Fahrzeugsicherheitssystem.

## Sicherheitsrelevante Hinweise

- Kein Einsatz im oeffentlichen Verkehr ohne weitere Validierung.
- Keine Ablenkung des Fahrers durch Displays oder Debug-Ausgaben.
- Warnungen duerfen nur als zusaetzlicher Hinweis verstanden werden.
- GPS-basierte Erkennung kann falsch-positive und falsch-negative Ergebnisse liefern.

## Meldung von Sicherheitsproblemen

Bitte keine sicherheitskritischen Details oeffentlich posten, wenn sie Missbrauch ermoeglichen koennten. Stattdessen ein privates Issue/Disclosure-Verfahren nutzen, sobald fuer das Projekt eingerichtet.
