# 09 Roadmap

## Version 0.1 / Konzept

- Konzept fuer Fahrer- und Fahrradfahrer-App.
- BLE/UWB-Analyse.
- Lokale Architektur ohne Cloud.

## Version 0.2 / GPS + Buzzer MVP

- ESP32 im LKW.
- GPS kommt von Smartphones.
- Lokale Risikoberechnung.
- Summer im LKW.
- iOS-Fahrer-App.
- iOS-Fahrradfahrer-App.

## Version 0.3 / Kalibrierung

- Testprotokolle.
- Konfigurierbare Warnschwellen.
- Mehrere Fahrraddatensaetze im ESP32.
- Stabilere Reconnect-Logik.
- Bessere Diagnoseausgabe.

## Version 0.4 / Hardwaregehaeuse

- Gehaeuse.
- 12/24-V-auf-5-V-Versorgung mit Sicherung.
- Buzzer-Lautstaerke.
- LED-Position.

## Version 0.5 / Genauigkeit verbessern

Optionen:

- BLE-RSSI als Zusatzsignal,
- externer rechter BLE-Scanner,
- UWB-SafetyTag,
- Smartphone-Kamera/Radar nicht im MVP,
- Karten-/Kreuzungserkennung.

## Version 1.0 / Feldpilot

- Test auf Betriebshof.
- Datenschutz- und Sicherheitskonzept.
- Protokollierte Testfahrten.
- Kein Einsatz im oeffentlichen Verkehr ohne Freigabe.
