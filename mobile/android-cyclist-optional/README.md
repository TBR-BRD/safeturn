# Android Fahrradfahrer-App

## Bauen

1. Projektordner in Android Studio oeffnen.
2. Gradle Sync ausfuehren.
3. Auf echtes Android-Geraet installieren.
4. Bluetooth/Standort erlauben.
5. `BLE senden starten` antippen.

## Zweck

Die App sendet jede Sekunde ein kompaktes BLE Manufacturer-Data-Paket an das LKW-Gateway.

## Hinweis

Die MVP-Version muss im Vordergrund laufen. Fuer echte Tests spaeter Foreground Service implementieren.
